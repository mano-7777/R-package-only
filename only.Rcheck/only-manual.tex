\begin{Description}\relax
More detailed description.
if there is any datetime column in the dataset and a step that you want to grab only the hour from the datatime column this package is useful to you.
\end{Description}
%
\begin{Usage}
\begin{verbatim}
only_hour(x)
\end{verbatim}
\end{Usage}
%
\begin{Arguments}
\begin{ldescription}
\item[\code{x}] Numeric variable
\end{ldescription}
\end{Arguments}
%
\begin{Value}
Factor variable
\end{Value}
%
\begin{Examples}
\begin{ExampleCode}
bike <- read.csv('bikeshare.csv')
head(bike$datetime)

only-hour<-sapply(bike$datetime,only_hour)

\end{ExampleCode}
\end{Examples}
